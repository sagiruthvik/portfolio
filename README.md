# Portfolio — Sagi Ruthvik

Dark, motion-heavy portfolio: ~10,000 WebGL particles assemble into the name, ripple away from the cursor and periodically swirl into a galaxy. Smooth scrolling, kinetic type reveals, a draggable 3D skill sphere, and a portal to the playground.

**Live:** https://sagiruthvik.github.io/portfolio/

## Stack

- [Vite](https://vitejs.dev/) — build tool
- [Three.js](https://threejs.org/) — particle hero
- [Lenis](https://lenis.darkroom.engineering/) — smooth scrolling
- Vanilla JS + CSS — everything else

## Develop

```bash
npm install
npm run dev
```

## Deploy

Pushing to `main` triggers the GitHub Actions workflow which builds and deploys to GitHub Pages automatically.
